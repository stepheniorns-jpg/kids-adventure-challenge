# Kids Adventure Challenge Starter

A scaffolded Expo + React Native starter for a family adventure challenge app.

## Included
- Expo Router navigation
- mock auth flow
- home + challenges + rewards + family + settings tabs
- Zustand family store
- Supabase client scaffold
- photo-proof hook
- GPS verification hook
- challenge completion flow
- initial SQL migration for Supabase

## Quick start
1. Install Node 20+.
2. In this folder, run `npm install`.
3. Copy `.env.example` to `.env` and add your Supabase values.
4. Run `npm start`.
5. Open in Expo Go or an emulator.

## Notes
- This is a starter scaffold, not a fully wired production build.
- Photo uploads and approvals are stubbed enough to demonstrate the architecture.
- Replace the mock sign-in logic with Supabase Auth before publishing.
